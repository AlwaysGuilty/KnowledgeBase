# ffmpeg

Usage: `ffmpeg -i <in> <out>`

Switches:
| Switch | Options | Description |
|-|-|-|
| `-i` | | input file |
| `-an` | | disable audio |
| `-vcodec` | `h264`, `hevc` | set video codec |
| `-acodec` | | set audio codec |
| `-b:v` | `1k`, `3M`, ... | set video bitrate |
| `-b:a` | | set audio bitrate |
